介紹


Malwarebytes Windows Firewall Control是一款电脑网络防火墙软件，该工具能加强系统本身防火墙系统，并且对指定程序添加windows规则的防火墙


Windows Firewall Control官方版是一款十分优秀的专业化网络防火墙，联网控制工具，Windows Firewall Control官方版能够为Windows用户提供更简单直观的防火墙使用方法的软件。当然你也可以设置过滤的规则，它令管理和应用Windows内置防火墙更加方便。

可勾选集成右键菜单,右键添加要阻止联网的程序
