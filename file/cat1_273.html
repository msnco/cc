介紹


《猎户座(Avorion)》是由Boxelware制作的一款模拟经营类游戏，游戏中玩家将进入程序生成的星系，操控太空船和空间站，去到没有任何人去过的地方。另外游戏中包含众多的内容，玩家可以建立自定义的太空船，建立舰队，参加贸易，开采资源，并加入史诗般的空间战斗中去。

几百年前，一场灾难性的灾难几乎将你的星系撕裂了——在星系中心出现了一个无法逾越的超空间结构环，这是普通的超空间引擎无法克服的。
自从这次事件以来，没有人设法靠近银河系的中心区域。

你所知道的是，该事件还在整个银河系中产生了多个无法穿越的超空间裂缝，一个奇怪的外星人种族，Xsotan，已经出现在中心。看起来这些外星人已经找到了穿越撕裂的超空间结构的方法，但到目前为止还没有人设法与他们建立联系。

也有传言称，在 Xsotan 到达的同时，一种名为“Avorion”的奇怪新金属出现在银河系的中心。显然，外星人使用这种材料来建造他们的船只。
从边缘的无名小卒开始，然后朝着一个更危险的星系中心前进，但越接近它的核心，奖励也就越多。 

Avorion 从 X 或 Freelancer 等游戏中汲取沙盒方面的元素，加入多人合作游戏并让您建造自己的飞船。它的特点是飞船由可自由扩展的方块组成，这些方块可以通过程序生成，并在太空战斗中被击中时会破碎。

注意事项：
1. 绝对不要在命名舰船、地图时输入中文字符(全角字符)，否则游戏将会崩溃
2. 尽量不要在除聊天窗口以外的其它地方输入中文字符，以避免可能产生的问题
已知问题：
1. 由于游戏自身问题，工厂、空间站的规模描述词会显示错误。如：南(小型)，M(中型)，L(大型)，XL(特大)，XXL(超大)
2. 由于游戏自身问题，极少数词语无法单独翻译。如：主菜单的“货币”实际为“开发人员名单”(开发组对两个词条均使用了“Credits”，无法翻译为不同词语)
3. 由于游戏自身问题，有时(通常为载入界面)部分词条显示为英文，属于正常现象
4. 由于游戏自身问题，已将默认字体替换为黑体，显示效果将于英文版游戏不同
5. 由于游戏自身问题及兼容性考虑，将不会汉化派系、人物、舰船名称等随机词语

设置中文
在游戏主菜单点击 Settings-Game，将 Language 选项切换为 Chinese_simplified
若下次进入游戏时变为英文，先将 Language 切换为 English，再切换为 Chinese_simplified
