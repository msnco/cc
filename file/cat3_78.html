ParagonNTFS


 Paragon NTFS For Mac是一款強大的Mac外掛程式，用戶可以直觀在Mac OS上 對外置NTFS格式 移動硬碟 或U盤中 的 檔，進行讀寫、添加、修改、刪除等操作，同時可以很好的保障資料的安全完整性。


軟體特色


1.完全讀寫NTFS


在Mac OS X中完全讀寫、修改、訪問Windows NTFS硬碟、U盤等外接設備的檔。


2.急速安全傳遞


高速傳輸外接NTFS硬碟檔，paragon ntfs for mac同時保障您資料的安全性和完整性。


3.簡單快捷操作


將硬碟或其它外置存放裝置與Mac相連，無需執行特定命令即可對NTFS分區進行載入。


4.超強相容性


完美相容Mac OS X所有版本、32/64位核心模式，及其它協力廠商軟體。


在Mac電腦上向Microsoft NTFS驅動器上寫檔


在Mac電腦上使用Microsoft NTFS格式的外部硬碟驅動器、SSD或拇指驅動器就像是使用本機磁片一樣。


全速讀寫您的所有目錄和檔。 掛載和卸載Microsoft NTFS卷、格式化分區，驗證磁片並檢查資料完整性。

讓您簡單自如讀寫NTFS外置存儲檔。在Mac OS X下全讀/寫訪問NTFS的任何分區，


實現在Windows 和Mac OS X電腦之間交替存儲和訪問檔，而無需重新格式化。僅在Mac電腦上安裝一次，便可讀/寫Windows相容格式的檔
