介紹


Green Screen Wizard Pro 是一款背景删除，绿屏摄影工具，

一款小巧专业的图像背景去除工具。可以帮您轻松去除照片中的人物或者其他杂物主体的背景，软件操作简便，用户只需要将人物在绿色背景布下简单点击，就可以轻松的去除人物或者杂物的背景了。

Green Screen Wizard Pro是一款功能强大的绿屏摄影软件，对于摄影爱好者与专业的摄影师来来说，对于绿屏摄影一定不陌生，摄影师进行绿屏拍摄后，可以通过绿屏摄影软件来进行后期处理，从而的得到各种令人惊叹的效果，而Green Screen Wizard Pro就是其中你的佼佼者，您可以实现添加各类背景、图像轮廓替换、色度键移除、图层叠加、喷枪编辑、批量打印等多种功能，而且具备了极其直观的操作界面，您可以通过简单几步完成处理，与其他处理软件相比简单了不少，而且功能十分的完善

智能图像处理
　　Green Screen Wizard Pro在处理图像时使用如此高的复杂程度，只需单击一下，即可立即生成十几个建议的调整。
　　直观的控制
　　Green Screen Wizard Pro可以精确地立即移除绿色屏幕，然后引导您完成背景替换过程。轻松微调图像或使用不同背景进行实验。
　　高级剪影
　　高级剪影可让您用任何颜色或图像替换轮廓。
　　皮肤光滑
　　通过平滑效果，让您的客户看起来更年轻甚至肤色。
　　添加反射
　　当您放置在水中或水中时，为您的主体添加逼真的反射。
　　最佳色度键移除
　　我们的专有算法使用惊人的精确度精确地从图像中删除绿色（或蓝色）屏幕 - 这使得Green Screen Wizard Pro软件更加出色。
　　令人惊叹的特效
　　使用新的纹理，反射，取景和柔化效果获得创意。
　　图层叠加
　　将您的拍摄对象置于“最多5个图层的场景中”。
　　喷枪编辑器
　　完全集成。触手可及的15把精美画笔。
　　Reality Integration
　　显着提高了Chroma Key去除的标准。
　　单个前景文件夹
　　在此过程中，图像被放置在前景文件夹中，并且对于前景文件夹中的每个图像，都有一个图像与绿屏向导的当前设置和背景相结合。
　　前景和背景文件夹
　　在此过程中，图像将放置在Foreground文件夹中，并指定背景文件夹。前景文件夹中的每个图像都与背景文件夹中的每个图像组合在一起。因此，例如，如果有10个前景图像和5个背景图像，则输出文件夹中将放置50个图像。
　　批量打印
　　批量打印允许您设置打印机和图像大小，然后打印图像文件夹。您也可以要求打印过程“继续检查新文件”，并且只要将图像放入要打印的文件夹中。
　　事件模式
　　事件模式与批处理类似，但它允许您检查并选择要处理的下一个图像。在事件模式下，您可以保存图像，打印图像或将图像保存到批量打印过程的组合图像文件夹。
　　批量处理
　　绿屏向导允许您获取两组文件，前景集和背景集，并将两者相乘。例如，如果您有五个背景和三个不同的前景，那么在完成该过程时您将拥有15个图像。
　　绿屏向导Pro存储在文件本身中创建文件的所有设置。请参阅前面的GSW文件说明。批处理过程可以通过三种方式使用这些文件。它可以拍摄当前图像，只是将当前设置更改为GSW文件中使用的设置。它可以拍摄当前图像并替换设置和背景，也可以拍摄当前图像并替换前景并替换。
　　批量打印
　　批处理可以创建数百个图像。创建批量打印过程是为了让您简单轻松地打印出这些图像。您可以通过选择输出打印机来启动此过程。接下来，选择要用于所有图像的打印机模板。您可以选择在页面上具有多个图像的模板。最后，选择要打印的图像文件夹。指示器将显示要打印的图像数量。只需单击开始，即可开始打印。
　　您可以观看新图像到达的文件夹。每次找到新图像时都会打印出来。图像可以通过GreenScreenWizard的其他副本或批处理过程放置在文件夹中，或者如果您绑定计算机，则使用网络共享软件。
