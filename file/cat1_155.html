驱魔道长 - 美少女cc | msncc

介紹


《驱魔道长》是由台湾厂商开发的一款新的中国风3DARPG游戏。 本游戏采用独特的3D引擎，并加入了原始的东方驱魔元素，带你穿越其中的奥秘。 坟墓，制裁恶鬼！ 致敬林正英捉鬼大师！
开发：Jaming Game Studio
发布：YIBAILI SOFTWARE
发布日期：2019.10.01
类型 : 动作、角色扮演、冒险
关于这款游戏
是一款ARPG动作游戏。 玩家将与茅山道人并肩作战，对付凶残嗜血的魔族，魔力变幻莫测的魔道，法宝如法！ 进入游戏，加入这场激动人心的驱魔之旅！
故事背景
北宋时，北辽国萧太后为报仇，拜道士陆克为军师，蜂拥而至 大军入宋。 吕克布设七十二道天门阵法，摧毁了龙虎山镇的魔神殿，放掉了当年被天师锁住的妖怪。 一些鬼怪逃到了人们身边，造成了不幸。
《驱魔道长》第一章，你是茅山道士阿楚，与毛小芳大师下山捉鬼，用智慧和魔法对抗鬼怪，找到失踪的妹妹 偏拼，最后与陆客的弟子华三道人在他所设立的邪坛上决一死战。
游戏特色
道士在鬼屋的密室里架起祭坛，香炉上放着通灵剑。 如果有鬼靠近，他们就会被剑杀死。
一般道士需要先招鬼才能捉鬼。 俗话说“半夜钟声，半夜鬼起”。 道士摇铃，将神界的鬼神引向阳界。
道士的主攻法术，踏足、掐诀、咏咒、背剑出鞘，此处剑光一闪，鬼已斩杀。
道士对幸存的幽灵施了一个恼人的咒语，以阻止其复活。
在密室的门窗上施咒，普通小子靠近就会死。
评论
游戏继承了东方传统文化，加入了驱魔道士的元素。 更加神秘诡异，更加激情澎湃，更加与众不同的原创剧情！ [高][高]
