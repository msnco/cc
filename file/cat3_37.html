Light Image Resizer 圖像壓縮處理軟件
 
光圖像大小調整是一個有用的應用程式，它可以用來調整，改變他們的決議或它們的大小您的圖片。
 
光圖像調整，是為那些誰商店的數碼照片和個人電腦的圖像和誰想要調整大小，壓縮，轉換，創建副本，進口或組織照片的最佳工具。
 
隨著光圖像調整，您可以創建您的圖片電子郵件友好的版本，載入它們更快，他們很容易從資料夾移動到資料夾，改變其格式，編輯大量影像檔，從而節省您的硬碟驅動器空間。
 
光圖像大小調整是集成在Windows資源管理器外殼，右鍵點擊你的圖片和你的照片開始工作！
 

 
 
 Light Image Resizer擁有很多不同的語言檔，當然也包括中文。調整圖片大小時，你可以添加一個檔，也可以添加一整個資料夾。Light Image Resizer的主要功能是：你可以通過設置百分比，或者圖片解析度來把圖像尺寸縮放到你想要的大小。並且通過保持寬高比，以及使用不同的縮放模式，可以讓圖片不變形
 
 當然，Light Image Resizer還是一個非常好的壁紙製作軟體，因為在它的設定檔中，默認就包含了iphone，sony psp，ipod螢幕解析度的檔，這樣你就可以非常輕鬆簡單的製作這些智慧移動設備的壁紙了。
 
 如果你覺的每次調整圖片都要手動到開始功能表中打開這個軟體很麻煩，那麼你可以在安裝的時候選擇讓Light Image Resizer在右鍵功能表中添加一個功能表項目，以後只要在圖片文章上右擊，選擇這個功能表項目就可以運行這個軟體了。
 
 當然，Light Image Resizer還有導入匯出設定檔功能。如果經常用這個軟體，那麼你一定不希望自己的設定檔在重裝後消失，而使用導入匯出功能可以讓你在任何一台電腦上都可以使用你自己的設定檔。
 
Light Image Resizer軟體功能
 
 調整圖片、照片、圖像大小，壓縮、轉換並輕鬆創建圖片副本;
 
 快速、批量調整照片大小;
 
 使用超快速多核心照片尺寸調整技術確保高品質結果;
 
 高品質圖像大小調整濾鏡，Lanczos 和 Bicubic 最佳壓縮品質以及線性速率;
 
 創建電子郵件友情圖像或者使用互聯網進行發佈分享;
 
 排序、重命名或者添加浮水印以保護圖片免於被盜用;
 
 支援使用設定檔庫跨平臺同步;
 
 支援創建自訂 PDF 頁面;
 
 簡易轉換 JPEG 圖像到 PDF 文檔;
 
 重新設計的使用者交互介面;
 
 創建用於列印或社交網路的若干圖像的貼圖;
 
 調整亮度與對比度、更改色彩深度(BMP 和 JPEG);
 
Light Image Resizer軟體特色
 
 1、擁有不同的縮放濾鏡，選擇不同可以在速度和品質之間找到你想要的平衡點。
 
 2、如果有些圖片小於設置的調整後的解析度，你可以設置在調整圖片大小時是否調整這些圖片的大小。
 
 3、你可以把所有的圖片轉換成許多中常見的格式。
 
 4、你可以選擇調整尺寸後的圖片品質，或者直接限制調整後圖片最大的體積。
 
 5、可以改變圖片的解析度(dpi)，通過更改圖片每一英寸擁有多少圖元，可以非常好的控制圖片的大小。
 
 6、可以給圖片加上浮水印，這個對於網站站長來說非常有用，使用LightImageResizer可以給網站上所有的文章圖片批量加上浮水印。並且還可以加上邊框和濾鏡效果。
 
Light Image Resizer支援格式
 
 支援輸出的格式：*.BMP、*.JPG、*.GIF、*.PDF、*.PSD、*.PCX、*.TGA、*.PNG、*.TIFF、*.WEBP;
 
 支援輸入的格式：*.JPG、*.CR2 (Canon RAW)、*.NEF (Nikon RAW)、*.DNG (Adobe)、*.GIF、*.ICO、*.PNG、*.PSD (Photoshop)、*.TIFF、*.ARW、*.BMP、*.CRW、*.MEF、*.CUT、*.DCM、*.DIB、*.DICOM、*.EMF、*.JP2、*.JPE、*.MRW、*.ORF、*.PBM、*.PCD、*.PCX、*.PEF、*.PPM、*.PSP、*.PSPIMAGE、*.RAF、*.RGB、*.SGI、*.SR2、*.SRF、*.TGA、*.TIF、*.WMF;
