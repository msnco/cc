诗酒剑江湖 - 美少女cc | msncc

介紹


《诗酒剑江湖》是款横版角色懒人养老游戏，诗词曲赋为题材，简单粗暴休闲风格，肝帝毫无用武之地，随机装备词条，强大的设置功能支持修改游戏文件重组游戏，包括技能特效和角色，大部分截图和字体音效序列帧动画素材都可直接替换使用，自动存档，优美方言配音。
名称: 诗酒剑江湖
类型: 休闲, 独立, 角色扮演
开发商: 我一人足矣
发行商: 我一人足矣
发行日期: 2021年4月9日

1.《自定义 UI》200 多个 UI 设置项目，多种字体字号 RGBA 颜色对不同部分文字显示分别设置，创造自己的 UI 界面，同时还可以把设置文件分享给他人使用。
2.《80 首诗词曲赋方言配音》参与秘境战斗时可以聆听到精选 80 首中国诗词曲赋，全部是方言配音，不一样的韵味。
3.《简单粗暴休闲玩法》属性玩法和各种游戏道具简化，只保留最精华的部分，去掉重复而没有独有特色的，让玩家体验真正的简单休闲好玩不坑不氪不肝。
4.《参与游戏设计》玩家可以参与游戏的设计，真正的体验当策划的感觉，如果你有关于游戏各种音效字体界面立绘动画特效玩法等相关的想法，请加 QQ 群 600733136。
5.《抽奖》抽奖玩法，玩家可以消耗副本掉落的道具进行抽奖，获得不同数量的同样的材料，小键盘数字 1 可以快速抽奖，来测试下人品了！
6.《随机词条》三个属性数值随机。数字小键盘 1 键可以快速做装备，可以嗨起来了！
7.《自动存档》可以随意关闭游戏，存档文件在根目录下可以复制使用。
8. 希望这个的小游戏能给大家带来几个小时的快乐，大家多参与，游戏越做越好！
